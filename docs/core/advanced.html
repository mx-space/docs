进阶部署

通过源码部署 Mix-Space 后端

在您进行进阶部署前，我们给予最后的警告：如果你不是开发者，或者不想折腾，请不要选择进阶部署，因为它需要你具备一定的开发能力。

由于自身技术原因导致的问题，我们将不会提供任何技术支持。情节严重者，我们将永久拉黑您的账号。

要求

  • 已安装 Node.js 20.0.0 或以上版本
  • 已安装 Git, PNPM, PM2
  • 已安装 MongoDB, Redis 并正常运行
